Accesibilidad

5.2.3 Técnicas para satisfacer los requisitos definidos en las WCAG

Técnicas Generales

  1. Añadir alternativas de texto a elementos multimedia.
  2. Permitir que el contenido pueda ser pausado.
  3. Ordenar el contenido de forma coherente.
  4. Añadir títulos descriptivos.
  5. Expandir abreviaturas.
  6. Eliminar parpadeos y destellos.
  7. Marcar la estructura de las tablas de datos(thead, tbody y tfoot).
  8. Proporcionar metadatos que den significado al documento.
  9. Utilizar atributo for del elemento label

Técnicas CSS

  1. Utilizar CSS para controlar el espaciado (margin y padding)
  2. No utilizar unidades relativas.
  3. Modificar los elementos al recibir el foco con el psudocase: focus.
  4. Utilizar valores porcentuales.
  5. Usar ojas de estilo vinculadas
  6. Utilizar propiedades como font-size.
  7. No utilizar elementos blink.
  8. Usar números en vez de nombres para especificar colores.
  9. Especifique un tipo de letra genérico por defecto.
  10. Asegurarse de usar colores con alto contraste entre el fondo y el primer plano.

Técnicas Aria

  1. Utilizar el atributo aria-label para etiquetar objetos.
  2. Utilizar el atributo aria-describedby en los controles de interfaz de usuario.
  3. Utilizar el atributo aria-required para campos requeridos.
  4. Utilizar el atributo aria-role para indicar el rol de un control y agrupar controles en grupos.
  5. Utilizar las aria landmarks para poder establecer regiones de una página.

Errores comunes

  1. No utilizar texto alternativos mediante alt.
  2. Usar textos alternativos muy largos.
  3. Publicar videos visuales sin alternativa de texto.
  4. Usar únicamente color para establecer jerarquías.
  5. Usar tamaño de los textos en medidas absolutas.
  6. No se indica el lenguaje de la página mediante el atributo lang.
  7. Entradas sin equiqueta label asociada.
  8. Imágenes decorativas no marcadas como tal.
  9. No informar al usuario de errores en la introducción de datos de entrada (aria-describedby).
  10. Utilizar links vacíos, sin atributo alt.
